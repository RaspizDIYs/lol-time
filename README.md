# LOLTIME

Приложение для мониторинга времени в League of Legends с мемными алертами.

## Функционал

- **Автоматическое отслеживание**: Следит за процессами `LeagueClient` и `League of Legends`.
- **Статистика**:
  - Время за сегодня
  - Время за неделю
  - Время за выходные (текущей недели)
  - Время за месяц
  - Общее время (Total)
- **Алерты**:
  - Настраиваемые лимиты (день, неделя, сессия).
  - Мемные уведомления (текст, картинка, звук).
  - "Жесткий режим" (спам уведомлениями).
- **Трей**: Работает в фоне, сворачивается в трей. Двойной клик открывает статистику.
- **Автообновление**: Встроена система обновлений (Velopack).

## Требования

- .NET 9 (SDK 9.0+)
- Windows 10/11

## Разработка

1. Собрать проект: `dotnet build`
2. Запустить: `dotnet run --project LolTime.App`

## Деплой (Создание релиза)

Для создания установщика и обновлений используется **Velopack**.

1. Установите Velopack CLI (один раз):
   ```bash
   dotnet tool install -g Velopack
   ```

2. Запустите скрипт сборки:
   ```cmd
   build_release.bat
   ```
   
   Это создаст папку `Releases` с установщиком (`Setup.exe`) и файлами обновлений.

## Настройка

При первом запуске приложение создаст файлы конфигурации в `%LocalAppData%\LolTime`.
Настроить лимиты и алерты можно через вкладку "Настройки" в главном окне.
